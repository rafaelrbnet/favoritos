# Produtos Favoritos

Projeto de uma API para produtos favoritos

* REST API  
CRUD de Clientes   
CRUD de Produtos   
Relacionamento de Clientes e produtos favoritos   
Autenticação via token

* Admin  
Sistema de administração das entidades Cliente, Produto e Favoritos
